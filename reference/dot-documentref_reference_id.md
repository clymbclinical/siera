# Extract a method's \`documentRef.referenceDocumentId\` (JSON shape)

Extract a method's \`documentRef.referenceDocumentId\` (JSON shape)

## Usage

``` r
.documentref_reference_id(document_ref, i)
```

## Arguments

- document_ref:

  The \`codeTemplate\$documentRef\` data frame from jsonlite.

- i:

  Method row index.

## Value

Scalar character id, or \`NA_character\_\`.
